Flight Network Dashboard – PDA Course Project

This folder contains an interactive HTML dashboard for the project:
"An Overview of Aircraft Flight Trends" (Programming for Data Science).

✅ To use:
1. Open the file: network_analysis_tabs.html in your browser.
2. Make sure the following files are in the same folder:
   - map_2019.html
   - map_2024.html
   - routes_2019_full.html
   - routes_2024_full.html
   - routes_2019_strong.html
   - routes_2024_strong.html

Each tab in the dashboard corresponds to a visual analysis of the European air traffic network for 2019 and 2024.

Enjoy exploring the network changes! ✈️
